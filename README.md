# SRI MEGHA AIRCON - Full Stack Website

Modern, responsive website for an HVAC service business. Built with React (Vite) and Node.js (Express).

## Features
- **Frontend**: React, Tailwind CSS, Lucide Icons, Glassmorphism design.
- **Backend**: Node.js, Express, integration with Google Sheets and WhatsApp.
- **Services**: Local SEO optimized, Contact forms, WhatsApp float button.

## Directory Structure
- `/frontend`: React application (front-end UI).
- `/backend`: Express server (back-end API & Integrations).

## Setup
### Prerequisite: 
1. Install Node.js
2. Clone this repository.

### Front-end Setup:
1. `cd frontend`
2. `npm install`
3. `npm run dev` (for development) or `npm run build` (for production).

### Back-end Setup:
1. `cd backend`
2. `npm install`
3. Create a `.env` file (see `.env.example` for details).
4. `node server.js`

## Google Sheets & WhatsApp Integration
To use these features, you must fill out the credentials in the `backend/.env` file.
